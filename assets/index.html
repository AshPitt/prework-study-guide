HTML

  • The head element contains informaiton about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML stands for HyperText Markup Language and is used to create web pages.
  • Tags are enclosed in angle brackets (< >) and can have attributes that provide additional information about the element.
  • HTML documents should start with a declaration that specifies the HTML version being used.
  • The element is the root element of an HTML document, and all other elements should be contained within it.
  • HTML has several types of elements, including structural elements (such as headings, paragraphs, and lists), text-level elements (such as bold and italic), and multimedia elements (such as images and videos).

CSS

  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Box model: Understanding how the box model works is fundamental to working with CSS. It describes how every element in a webpage is composed of a rectangular box that consists of four areas: content, padding, border, and margin.
  • Selectors: CSS selectors are used to select specific HTML elements on a page to apply styles to. There are several types of selectors, including element selectors, class selectors, and ID selectors.
  • Cascading and inheritance: CSS stands for cascading style sheets, meaning styles are applied in a cascading manner from top to bottom. Inheritance allows for the styles of a parent element to be inherited by its child elements.
  • Responsive design: With the rise of mobile devices, it's important for web developers to design responsive web pages that adapt to different screen sizes. CSS offers several tools for creating responsive designs, such as media queries and flexbox.
  • Box-shadow, border-radius, and gradients: These are some common CSS properties used to add visual appeal to web pages. Box-shadow can be used to add depth and dimension to elements, border-radius can be used to create rounded corners, and gradients can be used to create beautiful color transitions.
  • CSS preprocessors: Preprocessors like Sass and Less allow developers to write CSS with additional features like variables, functions, and nesting, making the code more modular and easier to manage.
  • Debugging tools: Debugging is a crucial skill for any developer. Chrome Developer Tools and Firefox Developer Tools are excellent tools for debugging CSS issues, as they allow developers to inspect the styles applied to each element and see how they interact with each other.

Git

  • Git is a version control system that allows you to track changes to files and collaborate with others on a project.
  • git init: Initialize a new Git repository
  • git clone: Create a copy of an existing Git respository
  • git add: Stage changes for a commit
  • git commit: create a new commit with the staged changes
  • git status: Show the current status of the repository
  • git log: show the commit history of the repository
  • git branch: create, list, or delete branches
  • git checkout: switch to a different branch or commit
  • git merge: merge changes from one branch into another
  • git pull: fetch changes from a remote repository and merge them into the current branch
  • git push: push local changes to a remote repository
  • git remote: manage remote repositories
  • git diff: show the differences between files or commits
  • git stash: temporarily save changes that are not ready to be committed
