HTML

  • The head element contains informaiton about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML stands for HyperText Markup Language and is used to create web pages.
  • Tags are enclosed in angle brackets (< >) and can have attributes that provide additional information about the element.
  • HTML documents should start with a declaration that specifies the HTML version being used.
  • The element is the root element of an HTML document, and all other elements should be contained within it.
  • HTML has several types of elements, including structural elements (such as headings, paragraphs, and lists), text-level elements (such as bold and italic), and multimedia elements (such as images and videos).

Git

  • Git is a version control system that allows you to track changes to files and collaborate with others on a project.
  • git init: Initialize a new Git repository
  • git clone: Create a copy of an existing Git respository
  • git add: Stage changes for a commit
  • git commit: create a new commit with the staged changes
  • git status: Show the current status of the repository
  • git log: show the commit history of the repository
  • git branch: create, list, or delete branches
  • git checkout: switch to a different branch or commit
  • git merge: merge changes from one branch into another
  • git pull: fetch changes from a remote repository and merge them into the current branch
  • git push: push local changes to a remote repository
  • git remote: manage remote repositories
  • git diff: show the differences between files or commits
  • git stash: temporarily save changes that are not ready to be committed
